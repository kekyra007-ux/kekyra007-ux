## Проекты

### Site Generator
Система генерации сайтов с использованием шаблонной архитектуры и AI-контента  
- Разработана система динамической сборки страниц на основе переиспользуемых шаблонов  
- Интеграция AI для генерации текстового контента и ускорения наполнения сайтов  
- Поддержка мультиязычных проектов и адаптации под разные рынки  
- Оптимизирована структура страниц под SEO и быструю индексацию  
- Сокращение времени запуска новых проектов и уменьшение ручной работы
  
[Смотреть проект](https://github.com/kekyra007-ux/nextjs-site-generator)


### E-commerce Store
Могостраничный интернет-магазин для портфолио на **Next.js + React + TypeScript + Redux Toolkit + styled-components**  
- Реализован каталог товаров с поиском, фильтрацией, сортировкой и пагинацией  
- Построена карточка товара с детальной информацией, related products и пользовательскими сценариями взаимодействия  
- Добавлены **wishlist** и **cart flow**: лайк товаров, добавление в корзину, изменение количества, удаление и пересчёт итоговой стоимости  
- Глобальное состояние организовано через **Redux Toolkit**: slices, async thunks, typed hooks и selectors  
- Использовано свободное API магазина для работы с каталогом и карточками товаров без необходимости регистрации
  
[Смотреть проект](https://github.com/kekyra007-ux/urban-nest-store)
